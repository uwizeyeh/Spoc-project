STUDY HOW TO PUT IMAGE

The image tag has two required attributes: src and alt. Note: Images are not technically inserted into an HTML page, images are linked to HTML pages. The image tag creates a holding space for the referenced image.

IS HOW TO PUT A LINK

HREF(Hypertext REFerence) The HTML code used to create a link to another page.
The HREF is an attribute of the anchor tag, which is also used to identify sections within a document. The HREF contains two components: the URL, which is the actual link, and the clickable text that appears on the page, called the "anchor text."
